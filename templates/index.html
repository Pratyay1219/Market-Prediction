Educational Stock & Commodity Price Prediction System

This web app demonstrates end-to-end predictive analytics using historical futures data for indices and commodities. It is designed for exams, interviews, and portfolio showcase – not for live trading.

What this project demonstrates

Data Preprocessing

Real EOD futures data processing and cleaning

Supervised Learning

Linear, Polynomial & Random Forest regression

Model Evaluation

MAE, MSE, RMSE, R² metrics

Time-Series Splits

Avoiding data leakage with proper validation

Neural Networks

LSTM for sequence modeling (experimental)

Bias–Variance

Comparison of simple vs. complex models

Use the Showcase page to browse HTML exports of the Jupyter notebooks, and the Prediction page to run models and compare classical regression against an LSTM sequence model.
